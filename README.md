## Run Locally

Requirements:

- Node.js 20+ (LTS recommended)
- Yarn

Install dependencies:

```bash
yarn
```

Start development server:

```bash
yarn dev
```

Build for production:

```bash
yarn build
```

Preview production build:

```bash
yarn preview
```

Run on local network (for phone testing):

```bash
yarn dev --host
```

Then open:

`http://<LOCAL_IP>:5173`
